## 2. 章節結構

| 代碼 | 作業名稱 | 章節說明 |
|------|----------|----------|
| CA101 | 固定資產取得作業 | 含新建、購置、捐贈取得、資產編號、資產卡、折舊與入帳 |
| CA102 | 固定資產異動盤點作業 | 含調撥、變更、增值、減值、分割、合併與年度盤點 |
| CA103 | 固定資產減損作業 | 含報廢、出售、盜失與減損處理 |
| CA104 | 投保作業 | 含取得後投保、保險標的與種類、保單管理與工程投保 |
| CA105 | 工程管制作業 | 含工程設計、發包、訂約、進度管制、監工、付款、異常、驗收與決算 |
| CA106 | 差異分析作業 | 含購置差異、維修費用、工程效益與成本效益分析 |

> 每個子作業均含作業程序、控制重點與相關資料。

## 3. 固定資產流程節點

```text
採購及付款循環
  → CA105 工程管制（工程設計 / 發包 / 訂約 / 監工 / 驗收 / 決算）
      ↓
  → CA101 固定資產取得（新建 / 購置 / 捐贈 / 編號 / 資產卡 / 入帳）
      ↓
  → CA104 投保（投保種類 / 金額 / 保單 / 續保 / 變更）
      ↓
  → CA102 固定資產異動盤點（調撥 / 變更 / 增減值 / 盤點）
      ↓
  → CA103 固定資產減損（報廢 / 出售 / 盜失 / 消失單 / 入帳）
      ↓
  → CA106 差異分析（購置成本 / 維修費用 / 工程效益 / 投資決策）
```

## 6. 主要控制點

### 固定資產取得階段

| 控制點 | 控制點類型 | ERP 落地確認 |
|--------|------------|--------------|
| 固定資產取得應依固定資產管理規定處理，使用單位申請新建、購置必經相關主管核准 | 權責控制 | 待系統確認 |
| 購買土地、房屋等不動產或大額固定資產，應依預算程序辦理並事前作效益評估 | 權責控制 | 待系統確認 |
| 房屋、土地等不動產取得或處分時，應先取得客觀獨立之不動產專業鑑價報告書 | 單據控制 | 待系統確認 |
| 固定資產依規定設立資產卡及明細帳，帳卡金額合計數應與總帳金額相符 | 單據控制 | 待系統確認 |
| 不可將資本支出列為費用支出或將費用支出列為資本支出 | 例外控制 | 待系統確認 |
| 固定資產應按期提列折舊，計算方法力求正確 | 系統控制 | 待系統確認 |

### 固定資產異動 / 盤點階段

| 控制點 | 控制點類型 | ERP 落地確認 |
|--------|------------|--------------|
| 固定資產異動非依規定程序辦理並填妥固定資產調撥單者，不得為之 | 單據控制 | 待系統確認 |
| 臨時緊急調撥須經主管同意，且在兩日內補辦調撥手續 | 例外控制 | 待系統確認 |
| 固定資產大修工程支出應注意有無超過預算並分析原因 | 系統控制 | 待系統確認 |
| 因經濟或效能問題轉列非營業固定資產時，若未達原計畫年限或投資回收數額，應檢討 | 例外控制 | 待系統確認 |
| 固定資產每年至少盤點一次，並注意漏盤、重複盤點與盤點卡連號控制 | 單據控制 | 待系統確認 |
| 單位主管調離職時，必須辦好單位資產卡移交工作 | 權責控制 | 待系統確認 |

| 工程成本效益應公正客觀比較與檢討，作為未來投資決策參考 | 權責控制 | 待系統確認 |

## 7. ERP 顧問訪談問題

### 固定資產取得 / 入帳
1. 固定資產新建、購置、捐贈取得目前由哪些單位申請與核准？
2. 固定資產編號規則、資產標籤與資產卡目前如何管理？
3. 固定資產取得後，殘值、使用年限、折舊方法與維修期間由誰維護？
4. 固定資產取得單與財會入帳目前是否有系統串接或人工轉入？

### 異動 / 盤點 / 保管
5. 固定資產調撥、移轉、閒置轉列、增減值與分割合併是否都有固定表單？
6. 固定資產盤點頻率、盤點範圍、盤點卡與盤點報告目前如何管理？
7. 單位主管離職或調職時，固定資產移交是否有清冊或系統流程？
8. 盤點差異或帳物不符目前如何追蹤與結案？

### 報廢 / 出售 / 盜失
9. 固定資產報廢、出售、盜失是否有不同核准流程與權限？
10. 未達耐用年限提前報廢時，是否需系統提示或特別核准？
11. 固定資產消失單、變更單與會計入帳目前如何銜接？

### 投保 / 工程
12. 固定資產投保項目、投保金額、保單到期與續保是否有系統提醒？
13. 工程概算、預算、發包、合約、監工、驗收與決算是否由同一系統管理？
14. 工程進度落後、品質異常、用料不符或付款異常是否需系統警示？
15. 工程完工轉固定資產時，固定資產增加單與發票 / 驗收資料如何連結？

### 差異分析
16. 固定資產維修費用、折舊、報廢損失與工程效益目前是否有分析報表？
17. 是否需要依資產類別、使用單位、品牌、地點比較維修費用與使用效益？
18. 固定資產資料是否要支援未來採購決策、投資效益比較或預算編列？
