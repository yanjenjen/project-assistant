## 五、章節級萃取欄位

第一份 DOC 的萃取草稿建議採用以下欄位：

| 欄位 | 說明 |
|---|---|
| 文件來源 | 原始檔名與 ZIP 內路徑 |
| 文件性質 | 內控制度 / 採購付款循環 / 通用範本 |
| 章節架構 | 只列章節名稱與順序 |
| 涉及 ERP 流程 | 請購、採購、驗收、付款、退貨、供應商等 |
| 主要控制點 | 以摘要方式整理，不貼全文 |
| 可能對應單據 | ERP 顧問可追問或建模的單據 |
| 待確認事項 | 原文不清楚或需使用者判斷的點 |
| 是否可沉澱 | 預設不可；需使用者確認後才可進入共用知識庫 |
