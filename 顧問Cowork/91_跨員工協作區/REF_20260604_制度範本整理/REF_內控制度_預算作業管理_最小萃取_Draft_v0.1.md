## 3. 預算作業管理流程節點

| 流程節點 | 作業摘要 | 後續 ERP 顧問觀察 |
|---|---|---|
| 預算組織建立 | 預算委員會由高階經理人組成，總經理擔任主任委員，總經理室主管擔任副主任委員與協調人 | 需確認預算組織、核准權限與跨單位協調角色 |
| 目標方針確定 | 經營階層提出年度目標方針計劃，董事會審核確認 | 需確認年度目標、預算版本與董事會核准流程 |
| 預算作業召集 | 預算委員會排定程序，召集各預算編製單位討論重點、程序與時程 | 需確認預算時程、通知與進度追蹤方式 |
| 銷售計劃編製 | 營業單位根據年度目標方針編製產品別、地區別銷售預算計劃 | 不推測 Sales 或 Forecast 對應，保留待系統確認 |
| 生產計劃編製 | 製造單位根據確認後銷售計劃編製生產數量、用料庫存、直接人工與製造費用等計劃 | 不推測 Manufacturing 或 MRP 對應，保留待系統確認 |
| 研發與管理費用編製 | 研發、企劃、財會、管理、總經理室等單位編製費用、人力、單位費用與資本支出計劃 | 需確認費用中心、成本中心與預算編製粒度 |
| 總經理室彙總 | 總經理室彙總審核各單位預算計劃，編製管理費用、資本支出、採購支出等預算計劃轉財會 | 需確認彙總層級、審核流程與轉交資料 |
| 財會彙編財務預算 | 財會根據各單位預算計劃彙編現金收支、營業外收支、預計損益表、資產負債表等 | 不推測 Accounting / Budget 對應，保留待系統確認 |
| 預算審核與實施 | 預算委員會審核整編，董事會審核，總經理室編製預算實施綱要 | 需確認核准版、執行版與發佈方式 |
| 預算檢討與修訂 | 每月比較預算與實際，每季檢討差異；特定重大變動可修訂預算 | 需確認差異門檻、追蹤責任與修訂版本管理 |

## 5. 角色 / 權責 / 簽核點

| 角色 / 單位 | 權責摘要 | 待確認事項 |
|---|---|---|
| 總經理室 | 本規章權責單位；彙總審核各單位預算計劃；編製年度預算實施綱要；每季召開預算執行檢討會並追蹤重大改善事項 | 總經理室與財會之預算資料分工需確認 |
| 預算委員會 | 決定經營目標及方針，審查初步預算，檢討修正事項，協調分歧，分析實際與預算差異 | 委員會正式成員與權限需確認 |
| 主任委員 | 由總經理擔任 | 是否為最終預算核准人需確認 |
| 副主任委員 / 協調人 | 由總經理室主管擔任，負責資料提供、彙總、建議、跟催與聯絡 | 協調人角色是否由單一人或部門承擔需確認 |
| 董事會 | 審核年度目標方針計劃、中長期計劃修訂與年度預算計劃方案 | 實際董事會核准節點需確認 |
| 營業單位 | 編製銷售計劃與銷售費用計劃 | 與銷售循環資料來源銜接需確認 |
| 製造單位 | 編製生產計劃、用料庫存、直接人工與製造費用等預算 | 與生產循環資料來源銜接需確認 |
| 研發單位 | 編製研究發展計劃與研發費用預算 | 與研發循環資料來源銜接需確認 |
| 管理單位 | 編製企劃、財會、管理、總經理室等單位費用預算 | 管理費用分類與成本中心需確認 |
| 財會單位 | 彙總編製現金收支、營業外收支、預計損益表、預計資產負債表，並每月提報預算實際比較 | 是否負責預算實際差異資料來源需確認 |

## 6. 主要控制點

### 權責控制

| 控制點 | 控制類型 | 系統確認狀態 |
|---|---|---|
| 預算委員會由高階經理人組成，並明確主任委員、副主任委員與成員角色 | 權責控制 | 待系統確認 |
| 預算委員會負責經營目標方針、初步預算審查、修正檢討、跨單位協調與差異分析 | 權責控制 | 待系統確認 |
| 年度目標方針與中長期計劃需經董事會審核確認 | 權責控制 | 待系統確認 |
| 預算委員會需召集各預算編製單位，溝通編製重點、程序與進度 | 權責控制 | 待系統確認 |
| 各單位經理需彙總並審核所屬預算編製單位之預算計劃，確認無誤後轉總經理室 | 權責控制 | 待系統確認 |
| 總經理室需彙總並審核各單位提報預算計劃，確認無誤後轉財會 | 權責控制 | 待系統確認 |
| 預算委員會應審核並整編所有年度預算計劃方案 | 權責控制 | 待系統確認 |
| 董事會應審核預算委員會提報之年度預算計劃方案 | 權責控制 | 待系統確認 |

### 單據控制

| 控制點 | 控制類型 | 系統確認狀態 |
|---|---|---|
| 年度預算計劃應以數據為主、文字為輔 | 單據控制 | 待系統確認 |
| 業務預算與財務預算下各項附屬計劃需明確列管 | 單據控制 | 待系統確認 |
| 依預算會議決議內容應發出書面通知，要求各單位開始作業 | 單據控制 | 待系統確認 |
| 各單位預算計劃應包含人力計劃、單位費用計劃、資本支出計劃等必要項目 | 單據控制 | 待系統確認 |
| 財會單位需根據各預算編製單位提報計劃，彙總編製現金收支、損益與資產負債等預算 | 單據控制 | 待系統確認 |
| 年度預算實施綱要應由總經理室根據董事會通過方案編製並送交各單位執行 | 單據控制 | 待系統確認 |

### 系統控制

| 控制點 | 控制類型 | 系統確認狀態 |
|---|---|---|
| 協調人需提供編製預算所需資料、彙總初步預算、跟催進度並督導執行 | 系統控制 | 待系統確認 |
| 預算編製體系需區分業務預算計劃與財務預算計劃 | 系統控制 | 待系統確認 |

## 7. ERP 顧問訪談問題

### 預算組織 / 權責

1. 預算委員會是否實際存在？成員、主任委員、副主任委員與協調人為誰？
2. 總經理室與財會單位在預算編製、彙總、實際比較與差異分析中的分工為何？
3. 預算編製單位是否以部門、成本中心、利潤中心或其他管理維度切分？

### 預算編製

1. 年度目標方針與中長期計劃如何形成、核准與通知各單位？
2. 銷售、生產、研發、管理、財會等單位各自使用哪些預算表？
3. 預算編製時程是否固定？是否有進度跟催與逾期提醒？
4. 各單位預算送出後，是否有版本控管與退回修正機制？

### 預算審核 / 實施

1. 預算委員會與董事會審核的是明細表、彙總表或完整年度預算方案？
2. 年度預算實施綱要如何發佈到各單位？
3. 核准版預算與各單位執行版預算是否一致？如何控管版本？

### 預算檢討 / 修訂

1. 財會每月比較預算與實際金額時，資料來源為何？
2. 差異達 10% 以上的判定基準是單月、累計、部門、科目或計劃別？
3. 差異分析與改善追蹤由誰負責？是否有結案標準？
4. 預算修訂需哪些條件、表單、核准流程與版本記錄？
