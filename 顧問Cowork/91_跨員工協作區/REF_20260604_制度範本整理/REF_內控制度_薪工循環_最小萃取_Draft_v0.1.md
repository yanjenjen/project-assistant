## 3. 薪工流程節點

| 流程節點 | 作業摘要 | 後續 ERP 訪談關注 |
|---|---|---|
| 人力資源規劃 | 依公司目標、經營計劃、勞工政策與人力供需規劃人員需求 | 是否有編制、職務、職能、職涯與人力需求預算資料 |
| 招募徵選 | 用人單位提出增補申請，人事依人力資源規劃決定內部晉升或對外招募 | 是否有招募申請、職缺核准、履歷、測驗、面試與錄取流程 |
| 任用報到 | 新進人員繳交證件、體檢、資料表、試用同意、勞保與扶養親屬資料 | 是否有新進報到清單、試用考核、任用權限與薪資級數流程 |
| 訓練研習 | 新進人員職前訓練、各單位提報教育訓練需求、人事彙整年度計劃 | 是否有訓練需求、課程、受訓紀錄、成績與派職升遷連結 |
| 考勤考核 | 打卡、補打卡、請假、出缺勤核對、績效評估與職工考績 | 是否有考勤、請假、加班、績效評估與薪資計算資料串接 |
| 獎懲升遷 | 依獎懲規定簽報、公告、記入人事資料，升遷依條件與程序辦理 | 是否有獎懲紀錄、晉升條件、調職與權限核准流程 |
| 薪資核計 | 依職工資料卡、調薪申請、出勤紀錄、加班、遲到早退、代扣款計薪 | 是否有薪資級距、調薪、加班費、扣款、勞健保與所得稅資料 |

| 福利管理 | 福利委員會制定福利辦法、管理福利金收支、辦理福利活動 | 是否有福利金、活動、會議紀錄與福利支出管理 |
| 離職資遣退休 | 離職慰留、交接、離職單簽核、資遣預告與資遣費、退休金計算 | 是否有離職流程、交接清單、離職給付、資遣退休條件與年資計算 |

## 5. 角色 / 權責 / 簽核點

| 角色 / 單位 | 權責摘要 | 待確認事項 |
|---|---|---|
| 財會單位 | 文件權責單位；涉及薪資登帳、代扣款、福利金與離職給付相關帳務 | 財會與人事於薪資核計的分工需確認 |
| 人事單位 | 招募、報到、試用、任用、訓練、考勤、考績、薪資資料、離職資料管理 | 是否有獨立 HR 系統與薪資系統需確認 |
| 用人單位 | 提出增補申請、面試、試用考核、績效評估、離職慰留與工作交接 | 用人單位主管核准權限需確認 |
| 各級主管 | 核准增補、請假、獎懲、調薪、升遷、離職等事項 | 權限層級與職等範圍需確認 |
| 計薪人員 | 依出勤、調薪、代扣款與薪資資料計算薪資 | 計薪作業是否由人事或財會負責需確認 |
| 會計 | 薪資核准後登帳 | 與出納發放流程銜接需確認 |
| 出納 | 薪資轉帳或現金發放、福利金出納控制 | 轉帳檔、現金領取與未領薪資處理需確認 |
| 職工福利委員會 | 擬定福利辦法、執行福利計劃、管理福利金收支與會議紀錄 | 是否納入 ERP 或另行管理需確認 |
| 總經理 / 董事會 | 依職級核准任用、教育訓練計畫、福利辦法等重大事項 | 實際核准層級需依企業規定確認 |

## 6. 主要控制點

### 人力資源規劃

| 控制點 | 控制類型 | 系統確認狀態 |
|---|---|---|
| 人力資源規劃須依公司目標、經營計劃、整體發展與勞工政策訂定 | 權責控制 | 待系統確認 |
| 人力資源規劃須定期更新，並考量升遷、福利與前置時間 | 系統控制 | 待系統確認 |
| 職業前程規劃應考量員工能力、個性與前瞻性，必要時採納員工意見 | 權責控制 | 待系統確認 |
| 人力冗員造成勞逸不均與挫折感時，應極力避免並檢討 | 例外控制 | 待系統確認 |

### 招募徵選

| 控制點 | 控制類型 | 系統確認狀態 |
|---|---|---|
| 增補人員應依規定填具申請並呈請核准 | 單據控制 | 待系統確認 |
| 員工須經審核或甄試合格後方得依規定聘用 | 權責控制 | 待系統確認 |
| 應徵文件表格須齊備，學經歷、技能、操守品德與健康狀況均需審查 | 單據控制 | 待系統確認 |
| 招募甄選方式應客觀公正，避免主觀印象與規定外承諾 | 例外控制 | 待系統確認 |

### 任用

|---|---|---|

| 各階層人員任用應依規定核准權限辦理 | 權責控制 | 待系統確認 |
| 應建立員工基本資料，敘薪標準依規定辦理 | 單據控制 | 待系統確認 |
| 試用不通過或不宜任用者應妥善處理，避免不必要損失 | 例外控制 | 待系統確認 |

### 訓練研習

|---|---|---|

| 各單位應依實際需要擬定教育訓練需求計畫並送人事彙整 | 單據控制 | 待系統確認 |
| 訓練研習應與人力資源規劃及人事政策配合 | 系統控制 | 待系統確認 |

## 7. ERP 顧問訪談問題

### 人力規劃 / 招募 / 任用

1. 人力需求是否有年度預算、編制或核准流程？
2. 增補人員申請如何發起、核准與追蹤？
3. 招募管道、履歷初審、測驗、面試與錄取通知是否有系統紀錄？
4. 新進報到需收集哪些文件？是否有報到清單與缺件追蹤？
5. 試用考核與正式任用是否與薪資級數連動？

### 訓練 / 考勤 / 考核

1. 教育訓練需求如何提報與核准？是否與預算連動？
2. 受訓成績是否納入人事紀錄、升遷或派職參考？
3. 出勤、補打卡、請假、加班與考勤異常如何處理？
4. 績效評估標準是否事先告知員工？考評結果如何保存？
5. 考勤與考績資料是否直接影響薪資、獎懲與升遷？

### 獎懲 / 升遷 / 調職

1. 獎懲案件如何提出、核准、公告與寫入人事資料？
2. 升遷條件、調職條件與權限核准是否已有明確規則？
3. 是否需要記錄獎懲佐證文件與審議歷程？

### 薪資 / 代扣 / 發放

1. 薪資級數、加給、加班費、遲到早退扣款與代扣款如何計算？
2. 調薪申請、核准與職工資料卡更新是否有完整流程？
3. 勞健保、所得稅、福利金等代扣款如何維護與核對？
4. 薪資發放採轉帳、現金或混合方式？是否有銀行轉帳資料與簽收紀錄？
5. 未領薪資、離職薪資與補扣款如何處理？

### 福利 / 離職 / 資遣 / 退休

1. 福利委員會、福利辦法、福利金收支是否需納入系統管理？
2. 離職流程是否需控管保管文件、資產、權限、工作交接與薪資結清？
3. 資遣預告、資遣費、退休金、平均工資與年資計算目前如何處理？
4. 離職原因是否會回饋人力資源規劃與招募需求？

## 9. 後續批次萃取格式建議

薪工循環可沿用既有內控制度最小萃取格式，10 節結構、第 4 節 6 欄單據格式與第 6 節四類控制點分類均可使用。

本次觀察到的格式議題如下：

| 議題 | 目前處理方式 | 是否需立即升版 |
|---|---|---|
| 人事、薪資、考勤、福利、離職等資料型態多元 | 仍以第 4 節「單據 / 表單 / 紀錄」方式容納 | 否 |
| 薪資代扣與法規資料可能需現行規則確認 | 放入待確認事項，不在草稿中下結論 | 否 |
| 請假與銀行轉帳資料名稱抽取不完整 | 以 `[待確認]` 標示，不補造 | 否 |

建議先完成本草稿格式自檢與使用者驗收，再決定是否更新制度範本總控。若後續多份循環都出現資料型態過多問題，再評估格式規則草稿是否升版。
