```
#頁面"最新消息"：
請依照以下需求調整「最新消息」功能，先盤點影響範圍與修改計畫，確認後再進行修改。

需求背景：
目前系統已有「最新消息」功能，但後台維護畫面與前台呈現方式需要調整。

一、後台目前問題
參考圖片"C:\Users\jenny.lu\Documents\建大輪胎專案\設計圖庫\J測試系統化面截圖\螢幕擷取畫面 2026-06-10-01 最新消息.png"
1. 後台「最新消息」維護畫面中，內文欄位顯示區域過窄，長篇文字被壓縮成很窄的欄位，不利於編輯與檢視。
2. 後台目前沒有「圖片上傳」欄位，無法為每一則最新消息上傳對應圖片。
3. 內文需要保留換行與段落格式，避免前台顯示時文字全部擠在一起。

二、後台調整需求
1. 調整最新消息表單畫面，讓「內文」欄位變成較寬的多行編輯區。
2. 新增「消息圖片」欄位，支援後台上傳單張圖片。
3. 圖片欄位需可供前台詳細頁顯示。
4. 既有欄位需保留：分類、主旨、發布日期、公司別、適用產品類別、啟用、內文

三、前台目前問題
參考範例："C:\Users\jenny.lu\Documents\建大輪胎專案\設計圖庫\J測試系統化面截圖\螢幕擷取畫面 2026-06-10-01 最新消息期望.png"
1. 目前前台「最新消息」頁面只顯示列表摘要。
2. 使用者點擊最新消息內容時，無法進入詳細頁查看全文。
3. 目前沒有顯示後台上傳圖片的位置。
4. 對方理想畫面希望可以看到「最新消息清單 + 全文內容 + 對應圖片」。

四、前台調整需求
1. 最新消息列表中的每一則消息需可點擊。
2. 點擊標題、卡片或「查看全文」後，需進入該則最新消息詳細頁。
3. 詳細頁需顯示：主旨、發布日期、分類、適用產品類別、完整內文、後台上傳的消息圖片、返回最新消息列表功能
4. 前台列表頁仍需顯示摘要，但摘要不取代詳細頁。
5. 若該消息沒有上傳圖片，詳細頁不可報錯，可不顯示圖片區或顯示預設圖片。

六、驗收條件
1. 後台最新消息內文欄位已加寬，可正常編輯長篇文字。
2. 後台可上傳最新消息圖片。
3. 前台最新消息列表可點擊進入詳細頁。
4. 詳細頁可顯示完整內文與圖片。
5. 內文換行格式可正常保留。
6. 沒有圖片的消息不會造成前台錯誤。
7. 前台只顯示符合登入者公司別、產品類別與啟用狀態的最新消息。
8. 調整不可影響既有商品總覽、訂單查詢、業績查詢、獎金查詢、對帳查詢、店招申請、合約下載、帳號管理等功能。
```
