# 【F003】最新消息功能改造 — 主需求

- **功能編號**：F003
- **功能名稱**：最新消息功能改造
- **提出人／日期**：【待確認】，內容來源 `230_內部專案檔案\建大系統調整 計畫表(提示詞).txt`
- **狀態**：已接受需求

## 背景

現有「最新消息」功能後台維護畫面與前台呈現方式不符使用需求：後台內文欄位過窄、無圖片上傳欄位；前台只顯示列表摘要、無法點入看全文與圖片。

## 主需求描述

**後台調整**：
1. 內文欄位改為較寬的多行編輯區
2. 新增「消息圖片」欄位（單張上傳）
3. 圖片供前台詳細頁顯示
4. 既有欄位保留：分類／主旨／發布日期／公司別／適用產品類別／啟用／內文

**前台調整**：
1. 列表項目可點擊進入詳細頁
2. 詳細頁顯示：主旨／發布日期／分類／適用產品類別／完整內文／消息圖片／返回列表
3. 列表頁仍顯示摘要，不取代詳細頁
4. 無圖片時詳細頁不可報錯

## 驗收標準

- [ ] 後台內文欄位已加寬，可正常編輯長篇文字
- [ ] 後台可上傳最新消息圖片
- [ ] 前台列表可點擊進入詳細頁
- [ ] 詳細頁可顯示完整內文與圖片
- [ ] 內文換行格式可正常保留
- [ ] 沒有圖片的消息不會造成前台錯誤
- [ ] 前台只顯示符合登入者公司別、產品類別與啟用狀態的消息
- [ ] 不影響既有商品總覽、訂單查詢、業績查詢、獎金查詢、對帳查詢、店招申請、合約下載、帳號管理等功能
